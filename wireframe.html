Section 4 — Curriculum Breakdown

Curriculum

9 modules — click to expand each:

Module 1: Foundations — AI, Automation & Agents
  • What is AI? (Narrow AI vs General AI, LLMs explained simply)
  • What is Automation? (Manual vs automated systems, real-world examples)
  • What is an AI Agent? (Chatbot vs Agent, components: Input, Brain, Memory, Tools, Output)
  • Understanding APIs (API keys, Webhooks explained visually)
Module 2: n8n for Absolute Beginners
  • What is n8n? (n8n vs Zapier vs Make)
  • Installing n8n (Cloud & self-hosted)
  • Core Concepts (Nodes, Triggers, Actions, Webhooks, Credentials)
  • Building Your First Workflow
  • Data Handling (JSON basics, Expressions, Mapping)
Module 3: Connecting AI (OpenAI / LLM Integration)
  • How LLM APIs Work (Prompts, System messages, Temperature, Tokens)
  • Connecting OpenAI to n8n
  • Prompt Engineering for Chatbots
  • Adding Personality & Role Control
Module 4: Telegram AI Chatbot
BotFather setup → n8n connection → OpenAI integration → Error handling → Memory. Final Project: Telegram AI assistant
Module 5: Instagram DM AI Chatbot
Instagram API & Meta Developer setup → Webhooks → AI processing → Auto-reply → Lead capture. Final Project: IG DM AI Sales Assistant
Module 6: WhatsApp AI Chatbot
WhatsApp Business API → Message handling → Structured replies → Compliance. Final Project: WhatsApp AI Support Bot
Module 7: Website AI Chatbot
Widget options → Webhook setup → AI processing → JSON responses → Styling & deployment. Final Project: Website AI chatbot
Module 8: Memory & Intelligence
Stateless vs Stateful → Google Sheets as memory → Supabase basics → Storing & retrieving conversation history
Module 9: Building Real AI Agents
Tool-using agents → Multi-step reasoning → Conditional workflows → External APIs. Final Project: Full AI Agent
